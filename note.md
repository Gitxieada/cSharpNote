###　快捷键:

```
C# Coding的时候，常常涉及代码注释，常见的注释包括两类：
1）单行注释。格式：// Comments
2）多行注释。格式：/* Comments... */
C#引入了新的XML注释，即我们在某个函数前新起一行，输入///

注释: Ctrl + Shift + /
保存单个文件：Ctrl + S
全部保存：Ctrl + Shift + S
显示剪贴板历史记录：Ctrl + Shift + V

启动调试 　　　　 F5
停止调试 　　　　 Shift+F5
```
